\section{Introdução}
    \subsection{Escopo}

    Este documento visa apresentar os recursos e a estratégia de desenvolvimento do projeto de Imobiliária, com os membros da equipe, custos totais, riscos que o projeto possa ter, previsão para o término e por fim recursos humanos e de tecnologia.
    
    \subsection{Objetivos}
    O projeto de DMCasa tem como objetivo ser um programa de Imobiliária, em que será possível alugar residências para ocupação. Assim será facilitado o monitoramento de imóveis, funcionários, clientes, gerentes e vistorias.
